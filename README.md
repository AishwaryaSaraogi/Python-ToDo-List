# To-Do List Script

A clean and efficient command-line Python script designed to help users manage daily tasks. This project demonstrates core programming concepts like list manipulation, loops, and user input validation.

## Features
* *Add Tasks:* Easily append new tasks to your list.
* *View Tasks:* Display all current tasks with their respective index numbers.
* *Delete Tasks:* Remove specific tasks from the list using their number.
* *Input Validation:* Handles invalid menu choices to ensure a smooth user experience.

## Technical Skills Demonstrated
* *Data Structures:* Using Python lists to store and manage data.
* *Control Flow:* Implementing while loops and if-elif-else logic.
* *Enumeration:* Using enumerate() for user-friendly indexing.

## How to Run
1. Ensure you have Python installed on your system.
2. Download the ToDoList.py file.
3. Open your terminal or command prompt.
4. Run the script using:
   ```bash
   python ToDoList.py
